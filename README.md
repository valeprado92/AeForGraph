# AeForGraph
Spatio-temporal graph autoencoder for automated evaluation of human actions in 3D in immersive VR-based training for archaeologists

This is the folder containing the scripts related to the paper "Spatio-temporal graph autoencoder for automated evaluation of human actions in 3D in immersive VR-based training for archaeologists".

Some of them deal with preprocessing, and the structure of the main script indicates the order in which to run them.
Finally, “variance_per_feature.py”, “statistical_analysis.py”, and “data_preprocessing.py” are not included in the pipeline because they are part of the preprocessing. “stream_and_response.py”, on the other hand, is used to connect Unity to Python.
